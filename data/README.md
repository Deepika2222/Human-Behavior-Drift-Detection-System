# Data Directory

This directory stores data files for the Human Behavior Drift Detection System.

## Structure:
- `raw/` - Raw, unprocessed behavior data
- `processed/` - Cleaned and preprocessed data ready for analysis and Extracted feature datasets
- `models/` - Trained model files and checkpoints
- `results/` - Drift detection results and analysis outputs

## Data Guidelines:
- Store large data files using Git LFS or external storage
- Keep sensitive data encrypted and follow privacy guidelines
- Document data schemas and formats
- Include data versioning information
- Add `.gitignore` entries for large or sensitive files
